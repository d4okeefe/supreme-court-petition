Template layout

The page layout here follows Supreme Court rules that specify briefs be printed on 6 ⅛ x 9 ¼ inch pages with ¾ inch margins.

Template design

This template has three parts: a cover, an index, and a brief. The parts are separated by section breaks, which you can see by clicking on the menu View > Show section breaks. The breaks will appear as dotted blue lines after the text on a page.

Text formatting

Text in Supreme Court briefs is set in a 12 point Century font with 14 point line spacing. Take care when pasting text from another source. You can unwittingly introduce fonts and font sizes that are not rule compliant.

Tip #1: When pasting text, try using the keyboard combination Control-Shift-v. The pasted text will match the formatting of the Google Doc paragraph.

Page numbers

The page numbers in the brief section are automated. However, the page numbers in the index must be manually updated.

By rule, index page numbers are numbered with Roman numerals (i, ii, iii). Google Docs only allows page numbering in arabic numberals (1, 2, 3), so you must manually type the index Roman numeral page numbers in the Google Doc header. We made a short video showing how to do this.

Table of contents

In this template, we include a Google Docs built-in Table of Contents, which you can view on page "v" of the index. However, the index page numbers will not auto-update.

We've provided a method for updating the index page numbers below. Before running, make sure that the Table of Contents is updated. (Update it by right-click anywhere in the TOC, then clicking "Update Table of Contents".)
